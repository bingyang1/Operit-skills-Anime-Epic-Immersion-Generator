 **Agent Skills / Prompt Framework**（可以称为“Anime Epic Immersion Generator” 或 “AEIG”），让AI在拿到任意一部动漫/轻小说/游戏/影视作品的核心信息后，尽可能复刻出类似结构、密度、沉浸感和质量的内容。

### AEIG v1.0 —— Anime Epic Immersion Generator
（高密度世界观 + 第一人称沉浸开局选择系统）

#### 核心设计原则（必须严格遵守）

1. **世界观必须同时浪漫化 + 残酷化**（绚烂却血腥、热血却绝望并存）
2. **查克拉/能量/魔力/灵压 等核心力量体系要放在显眼位置，并简要说明起源与三大/四大事物分类**
3. **宿命/轮回/对立理念传承** 是情绪最高点，必须提炼出两根（或多根）贯穿始终的“道路/意志/因果线”
4. **势力/组织/反派梯度** 要清晰（表面和平 → 暗中恐怖组织 → 最终外神/外星/神明级威胁）
5. **时间轴/时代划分** 用4–7个关键节点概括历史脉络
6. **尾兽/圣杯/恶魔/巨人之力 等战略级个体** 要列表化（编号 + 名字 + 持有者/归属 + 简短印象）
7. **重要NPC** 只列当前故事阶段最核心的5–10人，格式统一且带代表性标签（性格 + 信念/道路 + 关键外号/能力）
8. **开局选项** 必须提供 **8–10个** 差异化节点，覆盖：
   - 故事开端（第一话式）
   - 中期高潮篇章（考试/追杀/入侵）
   - 大战/最终章
   - 剧场版/外传/OVA风格
   - 续作/下一世代
   - 自定义槽位（最重要）
9. **语气与修辞**：大量使用热血、命运、羁绊、阴影、咆哮、刀锋、烈焰、泪水、轮回等意象词；句式短促有力 + 长句渲染氛围交替
10. **结尾触发句** 固定感召式：你的XXX，从这一刻点燃 / 你的传说，由此开始 / 你的命运，等你亲手书写！

#### 使用示例（极简版）

假如输入作品：《咒术回战》

你只需要提供（或让AI搜索）：

- 世界观一句话总结
- 核心力量：咒力 + 咒术 + 领域 + 术式
- 宿命对立：宿傩 vs 虎杖 / 呪术师 vs 人类 / 过去 vs 现代
- 主要咒灵/特级咒物列表
- 反派组织：脑花伪夏油 → 羂索 → 最终目的
- 核心角色5–8人
- 标志性篇章：起首、涉谷事变、死灭回游、涩谷后、真人篇、剧场版0等

然后AI就能按照上面9区块模板输出近似质量的内容。

| 要素覆盖度 | 预计相似度 | 备注 |

### AEIG v1.0 —— Anime Epic Immersion Generator
（高密度世界观 + 第一人称沉浸开局选择系统）

#### 使用示例（极简版）
(任意一部作品的名字 + 粗略核心设定即可生成高密度世界观 + 第一人称沉浸开局选择系统)
假如输入作品：《咒术回战》

你只需要提供（或让AI搜索）：

- 世界观一句话总结
- 核心力量：咒力 + 咒术 + 领域 + 术式
- 宿命对立：宿傩 vs 虎杖 / 呪术师 vs 人类 / 过去 vs 现代
- 主要咒灵/特级咒物列表
- 反派组织：脑花伪夏油 → 羂索 → 最终目的
- 核心角色5–8人
- 标志性篇章：起首、涉谷事变、死灭回游、涩谷后、真人篇、剧场版等
